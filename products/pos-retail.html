Logistics

Freight Calculation & Free Shipping Rules

Intelligent shipping cost engine that calculates freight based on location master data and applies free‑delivery thresholds automatically.

  • Freight Calculation

    Flat/fixed rate mapped against location master data (City/Province/Town). Automated calculation at checkout.

  • Free Shipping Threshold

    Automatically applies freight = 0 when order value exceeds Rs. 3,000. Encourages larger purchases.

  • Shipping Module

    Dedicated freight calculation and route charging. Supports multiple delivery zones and logistics partners.

Access Control

Security & Role‑Based Access

Granular Role‑Based Access Control (RBAC) ensures that every user has precisely the permissions they need — and nothing more. Secure your retail operations with confidence.

Permission

Define granular, page‑level permission sets with control over:

  • Read — View data and reports
  • Write — Create and update records
  • Edit — Modify existing entries
  • Delete — Remove records
  • Approve — Authorize transactions

Named permission profiles reusable across roles

Role

Create and manage system roles by bundling permission sets. Examples:

  • Store Manager — Full access to POS and inventory
  • Cashier — Process sales, issue returns
  • Auditor — Read‑only access to reports
  • Supervisor — Approve discounts and returns

Assign one or more roles to each user

User

Register and manage system users with:

  • User registration and profile management
  • Role assignment (single or multiple)
  • Active/Inactive status control
  • Password reset and account management

Link users to employee records (optional)

Every action is audited with user, timestamp, and IP logging for complete compliance and accountability.
